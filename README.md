**Backend Lead | Hackathon Project**

A comprehensive law firm management platform with advanced booking and file management capabilities.

**Key Features:**
- 🔐 JWT Authentication & bcrypt password hashing
- 📁 Secure file uploads with Multer + AWS S3 integration
- 📅 Advanced booking system with conflict resolution
- 👨‍💼 Admin dashboard with analytics
- 📧 Automated email notifications
- 🚀 Deployed on Render with environment optimization

**Tech Stack:** Node.js, Express.js, MongoDB, JWT, Multer, Nodemailer
